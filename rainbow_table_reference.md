# Rainbow Table Album Reference

## Complete Discography (Rainbow Table Only)

### Black - The Conjurer's Thread (2017)
- **Ontological Mode:** SPACE → TIME → INFORMATION
- **Concept:** Magic has stopped working, spiritualist's failed levitation
- **Key Track:** "The Conjurer's Thread" - mysterious ghost overtone at 03:51.043
- **Genres:** Art Rock, Experimental, Psychedelic, Post-Punk

### Red - Light Reading (2019) 
- **Mode:** PAST + THING + KNOWN
- **Concept:** Imagined reading list in song form
- **Key Tracks:** "A Doorbell for Finite Beings", "Dismantling Ubiquitous Monitoring Systems"
- **Theme:** Dead ideas, failed philosophers, forgotten knowledge

### Orange - Ruine (2020)
- **Mode:** PAST + THING + IMAGINED
- **Concept:** Broken memory palace, memory misused
- **Genre:** Alternative
- **Theme:** Personal memory to imagined futures

### Yellow - Pulsar Palace (2020)
- **Mode:** FUTURE + PLACE + IMAGINED
- **Concept:** Walking tour through cosmic manor, "equal parts grandeur and grotesque"
- **Genre:** Electronic, kosmische
- **Structure:** Each track = room in palace, BPM acceleration toward escape
- **Key Insight:** RGB strobing space where consciousness meets

### Green - The Empty Fields (2021)
- **Mode:** FUTURE + PLACE + FORGOTTEN
- **Concept:** Environmental collapse, ecosystem degradation
- **Genre:** New Music/Classical
- **Theme:** "Meditative, apocalyptic, and stoic with strange resilient beauty"

### Blue - Taped Over (2023)
- **Mode:** PRESENT + PERSON + FORGOTTEN
- **Concept:** Pre-digital nostalgia with sci-fi metaphysics underneath
- **Genre:** Folk Rock
- **Key Themes:** Alternate universe selves, digital/analog tension, Ship of Theseus identity

### Indigo - Infranyms (2024)
- **Mode:** FORGOTTEN + KNOWN (dual)
- **Concept:** Hidden information, decoder rings, "name below"
- **Genre:** Post-Rock
- **Theme:** Seven prismatic values, occult correspondences

### Violet - Vanity Pressing (2025)

## Cross-Album Patterns

### Temporal Progression
- Past (Red/Orange) → Present (Blue/Violet) → Future (Yellow/Green)
- Black/White exist outside linear time

### Consciousness Evolution
- Black: Loss of connection to higher purpose
- Red-Violet: Human experience across time/space/memory
- White: Information consciousness seeking embodiment

### Musical DNA
- Recurring themes: failed magic, memory palaces, alternate realities
- Technical elements: ghost overtones, cosmic timing, RPG structures
- Genre evolution: Art Rock → Electronic → Classical → Folk → Post-Rock → Pop
